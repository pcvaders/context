# Photodump — Style Presets Reference

Photodump is a Character tab feature that transforms a photo into a named artistic style.
Access: Character tab → Photodump (marked NEW)

Each preset applies a complete visual style — illustration type, rendering technique,
color palette, and cultural reference. "Use preset ✦" appears on hover.

## How to Use Photodump

1. Go to **Character tab → Photodump** (NEW)
2. Upload a photo (person, character, or scene)
3. Browse the preset gallery
4. Hover a preset → click **Use preset ✦**
5. The model applies the named style to your uploaded photo

## Prompt Tips for Photodump

- The preset does most of the stylistic work — minimal text prompt needed
- Focus prompt on: what the character should be doing, their expression, setting
- For more faithful character transfer: use a clear front-facing portrait as input
- For narrative scenes: describe the setting and action, let the preset handle the visual style
