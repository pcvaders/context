# Julius Brussee

## Background

Independent developer. Creator of the **caveman** Claude Code skill and surrounding ecosystem. Also built **Revu** — local-first macOS spaced repetition study app (FSRS algorithm). (as of 2026-05)

## Key contributions

- **[[caveman-julius-brussee|caveman]]** — output token compression skill for 30+ LLM agents. ~65% average reduction. Four intensity levels.
- **cavemem** — cross-agent memory system
- **cavekit** — spec-driven agent build loop
- **cavegemma** — Gemma 4 31B fine-tuned on caveman brevity pairs

## Philosophy

"Agent do more with less." Stack compresses what agents *say* (caveman), what they *remember* (cavemem), how they *build* (cavekit), and bakes compression into weights (cavegemma). Composable layers, one philosophy.
